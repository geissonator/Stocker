Welcome to Stocker!

The goal of this project is to provide a data repository containing all stocks and their associated data, a user interface to manipulate this database, and a generic plugin mechanism to generate stock selections algorithms based on that data.

Basic github setup

$ git init
$ git remote add origin git@github.com:geissonator/Stocker
$ git fetch origin
$ git checkout master

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.
